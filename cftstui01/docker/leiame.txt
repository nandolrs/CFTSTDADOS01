- Gerar build React

  cd C:\Users\073090631\Documents\nando\projetos\CFTSTDADOS01\cftstui01

  npm run build ..\.

- Gerar imagem UI

  docker build -f ..\Dockerfile -t cftstui-image ..\.
  
  docker images
		
- Teste rápido
		
  docker run -it --rm -p 3000:3000 --name cftstui-service cftstui-image

  acessar via navegador (chrome, edge, ...) = http://localhost:3000

  resposta = tela de pesquisa da aplicação

- Publicar no Docker
		
  docker tag cftstui-image nandolrs/cftstui-image

  docker image push nandolrs/cftstui-image
